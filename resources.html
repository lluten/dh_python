Pick one practice site, one reference, and one DH-focused resource to build a balanced study routine.

Python Practice and Basics

W3Schools Python

Use this for fast, practical exercises when you need to reinforce class topics like variables, loops, and lists. The short examples are great for quick revision before seminars.

Regex Help

Python re documentation

The official reference for Python regular expressions, including syntax, flags, groups, and function behavior.

RexEgg Quickstart

A compact guide to core regex syntax and common patterns. Keep this open while building extraction patterns for archival text or metadata fields.

regex101

Use this to test expressions against real sample text, inspect match groups, and debug mistakes line by line before moving patterns into Python code.

Digital Humanities Essentials

pandas Getting Started

Essential reference for tabular data work in DH, including CSV imports, filtering, grouping, and cleanup tasks common in catalog and corpus preparation.
