System Setup

Preparing your computer for Python-based Digital Humanities work or check out cloud-based alternatives.

Setup up your local development environment for Python-based Digital Humanities work.

Step 1: Install Python

Download the latest Python installer from python.org. Run the installer and ensure you check the box Add Python to PATH on the first screen.

Check the installation by opening a new terminal or PowerShell window and running:

POWERSHELL
python --version

Step 1: Install Python

You can use Homebrew to install the interpreter to avoid conflicts with the system Python:

TERMINAL
brew install python

Step 1: Install Python

Use your distribution's package manager and install the desired version. Check all available versions at python.org.

BASH
sudo apt update
sudo apt upgrade
sudo apt install python[VERSION]

Step 2a: Choose an Editor

Choose a development environment to write and run your Python code. You can either install a local editor or use a browser-based solution.

VS Code is a product of Microsoft and collects telemetry data. Similarly, Google Colab is operated by Google, collects usage data, and requires a Google account.

VS Code

A free, widely used code editor for local development.

Google Colab

A browser-based Python environment. No installation required.

Step 2b: Choose an Editor (Open Source)

If you prefer fully open-source software or university-hosted solutions, choose one of the following:

VSCodium

An open-source build of VSCode without proprietary components.

Jupyter Cloud

Browser-based Jupyter Notebook environment hosted by the university. No installation required.
