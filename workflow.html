Project Setup

Setting up structured projects and managing virtual environments.

Environments

A virtual environment is an isolated Python workspace with its own project-specific libraries. Creating a new environment for each project prevents dependency conflicts, ensures reproducibility, and keeps your code stable and easier to maintain.

Step 1: Create a Project Folder

Create a new folder for your project. You can do this in your file explorer or using the terminal:

TERMINAL
mkdir PROJECT_NAME
cd PROJECT_NAME

Step 2: Virtual Environment

In the project folder, create a virtual environment and activate it:

TERMINAL
python -m venv .venv
.venv\Scripts\activate
